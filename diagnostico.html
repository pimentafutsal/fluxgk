Diagnóstico de dados locais

Mostra o que o FluxGK guardou neste navegador. Não mexe no Firebase.

Importante: esta página só enxerga o navegador em que ela está aberta, e só se estiver no mesmo endereço do sistema. Abra-a a partir do mesmo lugar de onde você usa o FluxGK — se usa pelo site, acesse seu-site/ferramentas/diagnostico.html; se usa pela pasta, abra este arquivo pela pasta.

E os dados do Firebase?

São outra coisa, e esta página não os alcança. Ficam em users/{seu-uid}/data/{clube}/{temporada}/ e você os vê no Console do Firebase → Realtime Database → Dados.

A diferença prática: apagar o cache local aqui não apaga nada de verdade — na próxima abertura o sistema baixa tudo do Firebase de novo. Já apagar no Console do Firebase é definitivo.

Se um dado sumiu do Firebase mas continua aparecendo na tela, é cache local: limpe aqui.
Se aparece em dois clubes diferentes, é cache antigo de antes do isolamento por clube: limpe os itens marcados como legado.
